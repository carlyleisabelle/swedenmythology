Assignment Objective

The assignment objective is to create an interactive web page featuring a responsive map graphic. The map will be designed in Adobe Illustrator, converted to an SVG, and integrated with responsive HTML text. This project brings together cartographic design through the selection of a map theme and web development skills to create a functional and responsive web page web map.

About Sweden

This map highlights Sweden and its provinces, exploring the country’s deep-rooted folklore and mythology. Sweden, located in Northern Europe, is 450,295 square kilometers with a population of 10,040,995 people. Sweden is bordered by Norway and Finland and is so far North that in the summer, the sun never sets in the Northern Provinces (Sweden, 2014).

Sweden’s landscape is incredibly diverse, with mountains, dense old-growth forests, wetlands, and rolling countryside. Despite its size, the country is home to many national parks and nature reserves. People in Sweden have a deep cultural connection to the land, shaped by over 10,000 years of human history and a strong Norse and Viking heritage (Sweden, 2014).

Swedish Mythology and Folklore

Sweden has a long tradition of storytelling, with legends of mythological beings said to inhabit its wilderness. These tales, woven into Scandinavian and Nordic culture, have ancient origins and remain a strong part of Swedish identity today (Visit Sweden, 2022).

Each province has it’s own histories and mythological creatures, usually inspired by the natural environment of that area. Some of the most popular creatures include:

  • Huldra: An irresistible forest spirit, or the Scandinavian version of a siren (Visit Sweden, 2022).
  • Trolls: Giant, mischievous creatures that camouflage to their surroundings of caves and mountains (Visit Sweden, 2022).
  • Elves: Usually female fairy like spirits that live in the hills and come out to dance at dawn and dusk in the mist (Visit Sweden, 2022).
  • Draken (Dragons): With strong roots in Norse mythology, the dragon is a guardian of treasures and acts as a bridge between the living and divine realms (Johanna Hansen, 2025).
  • Tomte: Creatures with long white beards and pointy hats, similar to a gnome. Each Swedish home has one and it acts as a guardian spirit for the home and the people living in it (Johanna Hansen, 2025).

How to Make This Map

  1. Download or create an SVG image and open it in Adobe Illustrator.
    1. Add customizations to the map to align with the theme.
    2. Ensure all labels are point text not area text or objects.
    3. Use the fonts Arial or Georgia.
    4. Set the document colour mode to RGB.
  2. To convert the graphic to a responsive web map, a script called ai2html JavaScript code is used (New York Tiimes, 2024).
    1. Download the most up to date version of the script from ai2html.org.
    2. Install the script into the Adobe Illustrator application script folder.
  3. Once artwork has been completed, ensure Adobe Illustrator has been restarted and then run the script. After the first run is complete, a text box appears with various settings.
    1. Adjust the settings so that the image_format is SVG and responsiveness is set to dynamic. Optionally adjust the output location.
  4. Run the script again and now you will have an output in the output location selected. The output includes an html file and an SVG image.
    1. The SVG image can be moved into the img folder of the web map.
    2. The html document includes the css and html needed to view the formatted and dynamic labels. The html can be copies into the web page’s index.html file and the css can be copied into the style.css file.
      • To modify the style of the labels, changes can be made to the css code. This may include label size or colour.
      • To add hyperlinks to each label, navigate to the html and locate each label. Within the <p> tags of the label, <a> tags can be added with a url to link.

Refer to ai2html.org for more information and customization options.
